# SecureML-Pipeline

An end-to-end DevSecOps and MLOps project demonstrating a secure CI/CD
pipeline for machine learning model training and deployment.

## Structure
- ml/      : Model training and evaluation
- api/     : Model inference service
- ci/      : CI/CD and security pipelines
- infra/   : Kubernetes and infrastructure as code
